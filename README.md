# Merchant Digital Conversion Analytics

**Segment:** Merchant | **Category:** Revenue Optimisation | **Owner:** Straive Strategic Analytics | **Year:** 2024

## Objective
Analyse the checkout funnel end-to-end to identify drop-off points and quantify the GMV opportunity from digital wallet adoption and BNPL integration.

## Methodology
1. Session-level funnel mapping: cart → payment selection → auth attempt → approval
2. Digital wallet propensity model — identify customers likely to adopt
3. A/B test analysis for wallet promotion uplift
4. BNPL attachment rate and AOV lift measurement
5. Payment method P&L by channel
